# 12-01-PLAN: Early Story Quest Chains

## A) Goal
Create main story quest chains for Selva Esmeralda and Arrecife Hundido (Lv20-45).

## B) Context / Traceability
- **Requirement**: CONT-04
- **Why**: Ensures players have structured narrative content transitioning from the tutorial islands (Isla Brisa, Puerto Roca) into mid-game zones.
- **Dependencies**: Phase 11-01 (requires Items), Phase 11-04 (requires Monsters).

## C) Step-by-Step Implementation Instructions
1. **Open the File**: Modify `Content/Data/quests.json` or create a new arc-specific JSON file.
2. **Implement Selva Esmeralda Chain**:
   - Create 5-10 sequential quests for levels 20-30.
   - Establish `Requirements` linking so Quest N requires Quest N-1.
   - Objectives: Kill Jungle Goblins, collect Jungle Vines, interact with ancient totems.
3. **Implement Arrecife Hundido Chain**:
   - Create 5-10 sequential quests for levels 30-45.
   - Establish `Requirements` linking.
   - Objectives: Defeat Corrupted Merfolk, gather Sunken Pearls, defeat the Reef Boss.
4. **Rewards**:
   - Provide Tier 2-3 Equipment, Gold, and EXP scaling appropriately.
5. **ID Rules**:
   - Quest IDs must be 1000 or greater.
   - No dead ends: Ensure all `NextQuestId` or `Requirements` fields align correctly.

## D) Testing Requirements
- **Validation**: Ensure quest JSON is valid.
- **Test Suite**: Run `ContentValidationTests` to verify no duplicate Quest IDs and that all required items/monsters exist.
