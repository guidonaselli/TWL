# 11-02-PLAN: Tier 5-8 Items Foundation

## A) Goal
Append Tier 5-8 consumables, crafting materials, and late-game quest items (Lv45-100) to `Content/Data/items.json`.

## B) Context / Traceability
- **Requirement**: CONT-01
- **Why**: Completes the item progression database up to the max level cap (100), fulfilling the WLO-inspired 8-tier progression system.
- **Dependencies**: Phase 11-01.

## C) Step-by-Step Implementation Instructions
1. **Open the File**: Modify `Content/Data/items.json`.
2. **Implement Consumables**:
   - Add HP and SP recovery items for Tiers 5-8.
   - Scale `LevelRequired` (Tier 5: Lv60, Tier 6: Lv75, Tier 7: Lv90, Tier 8: Lv100).
   - Increase `BuyPrice` and `SellPrice` exponentially.
3. **Implement Crafting Materials**:
   - Add advanced ores (Mithril, Adamantite, Orichalcum, Starmetal) and woods.
   - Add advanced elemental cores/crystals.
4. **ID Rules**:
   - Maintain the 1-9999 ID space without colliding with Tier 1-4 items.

## D) Testing Requirements
- **Validation**: Ensure `items.json` remains valid JSON.
- **Test Suite**: Run `ContentValidationTests` to verify no duplicate IDs have been introduced.
