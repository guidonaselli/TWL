# SKILLS ROADMAP

## Definición de “listo” (DoD)
### Contenido (skills)
- Definición en JSON con schema y validación.
- Coverage mínima (si es skill pack: 3 ramas).
- Cero nombres/listas copiadas de WLO/NWLO/Wonderland M (Excepción: Nombres de Goddess Skills).
