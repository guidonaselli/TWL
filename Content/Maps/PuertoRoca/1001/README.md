# Puerto Roca Port (Map 1001)

## Overview
A busy port adjacent to Puerto Roca City. Connects travelers to the outside world.

## Layout & Triggers
- **Map Size:** 20x20 tiles (640x640px).
- **Triggers:**
    - **To_City (ID 1):** Leads to Map 1000 (City), TargetSpawnId 2. Located at East Center.

## Spawns
- **PlayerStart (ID 1):** Arrival point from City.
- **NPC (ID 2):** Ship Captain.

## Tileset Requirements
- Uses "placeholders" tileset.
