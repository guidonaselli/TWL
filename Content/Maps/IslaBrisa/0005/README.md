# 0005 - Mirador del Faro

**Region:** Isla Brisa (Starter Island)
**Type:** Landmark / Scenic View
**Biome:** Tropical (Cliffside)
**Level:** 3-5

## Description
A high cliff overlooking the ocean, featuring the ruins of an ancient lighthouse ("El Faro"). From here, players can see Puerto Roca in the distance, foreshadowing the next region.
