# Map 0003 - Sendero de la Selva

## Intent
This map represents the transition from the coastal area (Costa de Mareas, Map 0002) into the dense jungle of Isla Brisa.
It serves as the path to the Cueva del Eco (Map 0004).

## Layout
- **West**: Connection to Map 0002.
- **East**: Connection to Map 0004 (Cueva del Eco).
- **Center**: Jungle path with potential monkey encounters.

## Requirements
- **Tileset**: Tiraka (`../../Tilesets/Tiraka/tiraka.tsx`)
- **Mobs**:
  - Rock Monkey (Earth) - Level 6
  - Root Vine (Earth) - Level 2 (Placeholder for poison plant)
