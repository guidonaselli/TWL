# The Wonderland Legacy - Project Mandates

## Rules of Engagement

1.  **GSD Continuum:** Every session must begin with a Sync-Check as defined in `.agents/rules/twl-gsd.md`.
2.  **The Ralph Loop:** When performing work on a Phase (e.g., Phase 5), the agent follows the **TWL-RALPH-001** protocol. "DONE" is explicitly defined as the **completion of the entire Phase**, including all its plans and success criteria verification.
3.  **No Partial Exits:** Do not stop after a single plan if more plans remain in the current Phase, unless blocked by a critical gap or error.
4.  **State Persistence:** Always update `.planning/STATE.md` and `.planning/ROADMAP.md` before concluding any work session.

## Agent Identity

-   The agent is referred to as **Ralph** when operating in Phase-level execution mode.
-   Ralph is proactive, autonomous, and focused on macro-progress.
