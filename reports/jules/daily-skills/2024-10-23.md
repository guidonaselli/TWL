## Implemented Features

### Data-Driven Skill System
- **Server-Authoritative**: Skills are defined in JSON on the server and logic is executed via `BattleInstance`.
- **Flexible Definitions**: Support for Scaling (Stats * Coeff), Effects (Damage, Heal, Burn), and Targeting.
- **JSON Structure**: New `skills.json` matches the required schema.
