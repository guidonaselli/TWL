# Daily Skills Report - 2024-10-18

## Overview
Implemented a coherent package of 3 new skills demonstrating advanced WLO-like mechanics, along with necessary server-side infrastructure updates.

## Infrastructure Updates
### CombatManager & ServerCombatant
- **Cooldowns**: Implemented transient cooldown tracking in `ServerCombatant`. `CombatManager` now enforces cooldowns and sets them upon skill usage.
- **Outcome Model**: Implemented `Immunity` (Resistance >= 1.0) and `Partial` (Half Duration/Value) outcomes in `CombatManager`.
- **Resistance Logic**: Updated to check multiple `ResistanceTags` against target resistances.

### AutoBattleService
- **Dispel Heuristic**: Added logic to prioritize `Dispel` skills when enemies have beneficial buffs.
- **Cooldown Awareness**: AI now checks if a skill is on cooldown before selecting it.

## New Skills
1. **Boulder Crash** (Earth / Magical / T2)
   - **Effect**: Damage + Seal (Stun).
   - **Mechanics**: `Outcome: Partial` (Halves duration if resisted but not immune), checks `EarthResist` and `StunResist`.
   - **Unlock**: Parent Skill `Stone Bullet` (Rank 10).

2. **Gale Slice** (Wind / Physical / T1)
   - **Effect**: Damage + Burn (Bleed).
   - **Mechanics**: `StackingPolicy: StackUpToN` (Max 3 stacks), `ConflictGroup: DoT`.
   - **Unlock**: Level 5 + Stats.

3. **Aqua Shield** (Water / Support / T1)
   - **Effect**: Def Buff + Fire Resistance Buff.
   - **Mechanics**: `StackingPolicy: RefreshDuration`.
   - **Unlock**: Level 5 + Stats.
